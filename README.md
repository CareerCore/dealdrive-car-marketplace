# DealDrive

DealDrive is a premium car listing and buying platform built for the Pakistani market. It lets users explore a curated collection of cars — from budget SUVs to exotic supercars — all in one place.

## What it does
- Browse cars by category (SUV, Sports, Luxury, Electric)
- Search and filter by brand or price range
- Save favourite cars (stays saved after refresh)
- Add to cart and place orders
- Pay via Card, JazzCash, EasyPaisa, Bank Transfer, or Cash on Delivery
- Send a direct enquiry to our team
- Fully responsive — works on mobile and desktop

## About the Project
This project was built as part of a university assignment to demonstrate front-end web development skills. It features a cinematic video hero section, animated stats, a multi-step checkout flow, and a clean car catalogue with real specs and pricing in PKR.

## Cars Available
12 cars across 9 brands including Toyota, Tesla, Porsche, BMW, Range Rover, Audi, Honda, Hyundai, Haval, and Kia.

## Built With
- HTML, CSS, JavaScript
- Bootstrap 5
- jQuery

## How to Run
1. Download or clone the repo
2. Open `index.html` in your browser
3. Done — no installs needed
